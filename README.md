# Python-Projects
This repository contains the project build on python concept.

This project is about the **Login System** 

In this project:-
* I am first create a variable **Username** and **Password** in  which i store the original values.
* Then when user vsit again on this site he/she have to fill credential again.
* In this user have 3 chances to fill the credential.
* if he/she can fill the correct credential in 3 chances then they geta message **Access Granted**.
* And if they are not able to fill it in 3 chances they will get an Message **System has locked**
